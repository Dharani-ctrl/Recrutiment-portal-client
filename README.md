# RecruitMe – Recruitment Portal System

RecruitMe is a modern full-stack recruitment portal developed using the MERN stack (MongoDB, Express.js, React.js, Node.js). The platform connects companies and candidates through a seamless recruitment process with secure authentication, job management, and application tracking features.

## 📌 Features

### 👨‍💼 Admin / Recruiter
- Recruiter authentication
- Create and manage job postings
- View candidate applications
- Dashboard management
- Secure backend APIs

### 👨‍🎓 Candidates
- User registration & login
- Browse available jobs
- Apply for jobs
- Responsive UI experience

### 🔒 Authentication & Security
- JWT Authentication
- Protected Routes
- Password Encryption

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/Dharani-ctrl/Recruitment-portal-client.git
```

---

## 🔧 Frontend Setup

```bash
cd client
npm install
npm run dev
```

Create `.env` file:

```env
VITE_API_URL=http://localhost:5000
```

---

## 🔧 Backend Setup

```bash
cd server
npm install
npm start
```

Create `.env` file:

```env
PORT=5000
MONGO_URI=your_mongodb_connection
JWT_SECRET=your_secret_key
```
